# homebrew-tap

Homebrew formulae by [pacifistazero](https://github.com/pacifistazero).

## Usage

```sh
brew tap pacifistazero/tap
brew trust pacifistazero/tap   # recent Homebrew requires trusting third-party taps
brew install vdisplay
```

## Formulae

- **[vdisplay](https://github.com/pacifistazero/vdisplay)** — virtual displays and
  monitor-layout management for macOS.
